# Day 17 — Dice Roller Simulator

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A dice roller simulator with multiple dice types,
roll history and probability stats.

## How to view
Open `index.html` in your browser.

## Features
- Roll D4, D6, D8, D10, D12, D20 dice
- Roll multiple dice at once
- Roll history log
- Total and average stats
- Animated roll effect
